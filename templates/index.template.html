Purity in Coding

Simple, fast, functional programming. Crisp (for reduced lisp in C#) reduces programming to its purest form, then builds it right back out again. A clean and minimalist Lisp dialect based loosely on Lispkit Lisp with a small but beautiful standard library, Crisp takes programming back to its roots.

Expressive

Use the cleanest, most minimal lispy syntax to create your programs. Build fast, predicatable and efficient programs using only a few primitive data types (booleans, strings and numerics), functions and lists.

You've got it! Now what?

Before you get carried away! Crisp is in pre-alpha and is far from stable or fully optimised. Please do create cool stuff with it, but take care and be safe! It's not a good idea to use Crisp or Packet in a production environment or (heaven forbid) in any security/safety/business-critical system. - Saul Johnson

Need some pointers on getting started? You might want to check out what Crisp is or our introduction to the language itself. Once you're familiar with all that, it's probably time to get started. But keep the documentation handy to get off to the best start you possibly can! If you're a C# programmer and you need some new functionality, why not look in to writing an extension to the language?
