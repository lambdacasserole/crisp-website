Extending Crisp

If you can program in C# or VB.NET, you can write your own special forms to expose new functionality for your Crisp programs to take advantage of! For example, you might create a send-email special form to power your website contact form.

Creating a new special form really this simple:

  • Open Visual Studio, start a new class library project.
  • Add a reference to the Crisp.Core assembly
  • Add a new class, make sure your class inherits from Crisp.Core.Types.SpecialForm as its direct parent class.
  • Name your special form, The value of the name property determines the name used to call your special form from your Crisp source code.
  • Create your Apply method. The arguments are passed into the first parameter of the Apply method from the executing Crisp code, and from there it's up to you what you do with them!
  • Compile your class library and add it to the folder named native in the same directory as the crisp.exe or packet.exe executables.
  • Use your new special form from your Crisp code!

If you want to use Crisp to program for the web, it's worth checking out the Packet webserver which is specifically designed to serve dynamic webpages generated in Crisp!
