API Documentation

This page is intended as a reference for new Crisp programmers to help them become familiar with the language's core features. As the size of the Crisp standard library grows, so will this page!

Special Forms

Special forms are functions written in a .NET language (usually C# or VB.NET) that form the backbone of Crisp, exposing special functionality. To read more about how special forms are created and structured, check out the page on extending the language. A few binaries containing special forms core to Crisp's functionality are included by default and documented below (grouped by assembly).

Crisp.Basic

The Crisp.Basic.dll assembly contains the bare-bones core special forms that give Crisp its basic functionality. In here, you'll find the special forms that give Crisp basic Lispkit Lisp functionality.

Name Example Result Description
add (add 4 8) 12 Adds two numeric atoms together and returns the result as a new numeric atom.
atom (atom "strings are atoms") true Returns a true boolean atom if the given expression evaluates to an atom. Otherwise returns a false boolean atom.
car (car (cons 5 4)) 5 Returns the head (first) of a cons cell (pair).
cdr (cdr (cons 5 4)) 4 Returns the tail (rest) of a cons cell (pair).
cons (cons 9 8) (9 . 8) Returns a cons cell (pair) containing as its head and tail the two expressions given.
div (div 12 6) 2 Divides a numeric atom by another and returns the result as a new numeric atom.
eq (eq 12 (add 6 6)) true Returns a true boolean atom if the two given arguments are considered equal. Otherwise returns a false boolean atom.
if (if (eq 12 (add 6 6)) "foo" "bar") "foo" A function that returns its second parameter if its first parameter evaluates to a true boolean atom. Otherwise returns its third parameter.
lambda (lambda (x) (add x 1)) λ-expression Returns a callable function (closure) with the specified parameters and body.
leq (leq 4 6) true Returns a true boolean atom if the first argument evaluates to less than or equal to the second argument. Otherwise returns a false boolean atom.
letrec (letrec (add x y) (x . 10) (y . (add x 5))) 25 A function that binds symbols to expressions, allowing for recursion.
let (let (add x y) (x . 10) (y . 5)) 15 A function that binds symbols to expressions.
list (list 1 2 3) (1 . (2 . (3 . nil))) A function that returns its arguments as a proper list.
mul (mul 6 10) 60 Multiplies a numeric atom by another and returns the result as a new numeric atom.
prog (prog (file-set-text "C:\foo.txt" "bar") (sub 10 5)) 5 A function that sequentially evaluates its arguments, returning the result of evaluation of its last argument. Useful for programming with side-effects.
quote (quote foo) foo A function that will convert all symbols in an expression to constants.
rem (rem 10 3) 1 Divides a numeric atom by another and returns the remainder as a new numeric atom.
sub (sub 10 3) 7 Subtracts a numeric atom from another and returns the result as a new numeric atom.

Crisp.Data

The Crisp.Data.dll assembly contains special forms designed to work with the bundled 100% managed port of the SQLite serverless, zero-config database engine. This assembly is currently very minimal, but should be sufficient for most database tasks.

Name Example Result Description
data-non-query (data-non-query "data.db3" "INSERT INTO Employees (name, age) VALUES ('Fred', 25)") 1 Executes an SQLite non-query against a database file, returning the number of affected rows.
data-query (data-query "data.db3" "SELECT * FROM Employees") ((("name" . "Fred") . (("age" . "25") . nil)) . nil) Executes an SQLite query against a database file, returning a set of name-value collections representing rows retrieved.

Crisp.IO

The Crisp.IO.dll assembly contains special forms designed to work with the file system. For example, reading and writing text files and checking whether or not directories/files exist on-disk.

Name Example Result Description
directory-exists (directory-exists "C:\this_folder_exists") true Returns a true boolean atom if a directory exists at the specified path. Otherwise returns a false boolean atom.
file-exists (file-exists "C:\this_file_exists.txt") true Returns a true boolean atom if a file exists at the specified path. Otherwise returns a false boolean atom.
file-get-text (file-get-text "C:\hello_world.txt") "Hello world!" Reads all text from a file and returns it as a string atom.
file-set-text (file-set-text "C:\foo_bar.txt" "foobar") true Writes the given text to the file at the specified path and returns a true boolean atom on success. On failure returns a false boolean atom.

Crisp.String

The Crisp.String.dll assembly contains special forms designed for string manipulation. This is currently far from a comprehensive set of string manipulation functions but will suffice for many use-cases.

Name Example Result Description
chr (chr 65) "A" A function that converts from a numeric ASCII code to its character equivalent.
concat (concat "he" "llo") "hello" Returns a new string atom consisting of two string atoms concatenated together.
html-decode (html-decode "&lt;hr&gt;") "<hr>" HTML-decodes a string atom and returns the result.
html-encode (html-encode "<hr>") "&lt;hr&gt;" HTML-encodes a string atom and returns the result.
regex-replace (regex-replace "0f37d6" "[0-9]" "#") "#f##d#" Returns a string atom consisting of the first argument passed to this function with all matches to the regular expression specified as the second argument replaced with the string passed as the third argument.
replace (regex-replace "hyphenated-string" "-" " ") "hyphenated string" Returns a string atom consisting of the first argument passed to this function with all occurrences of the string specified as the second argument replaced with the string passed as the third argument.
split (split "10.11.12" ".") ("10" . ("11" . ("12" . nil))) Returns a list of substrings created by splitting a string along a delimiter.
url-decode (url-decode "hello%3dworld%26+foo%2fbar") "hello=world& foo/bar" URL-decodes a string atom and returns the result.
url-encode (url-encode "hello=world& foo/bar") "hello%3dworld%26+foo%2fbar" URL-encodes a string atom and returns the result.

Standard Library

The Crisp standard library is written in Crisp and contains useful functions composedfrom the special forms defined in C#.

core.cspl

The core.cspl library contains some of the most commonly used functions inCrisp for general programming.

last

Returns the last element in a proper list.

Type Parameter Description
list lst the proper list to return the last element of

concat-all

Concatenates a list of string expressions into one string expression.

Type Parameter Description
list lst the list of string expressions to concatenate

null

Returns true if the given paramters is nil.

Type Parameter Description
any x the expression to check

and

Returns true if both parameters evaluate to true.

Type Parameter Description
bool x the first argument
bool y the second argument

or

Returns true if one or both parameters evaluate to true.

Type Parameter Description
bool x the first argument
bool y the second argument

xor

Returns true if one input (but not both inputs) evaluates to true.

Type Parameter Description
bool x the first argument
bool y the second argument

append

Appends an atom or list to another list.

Type Parameter Description
list lst the base list
list x the list of atoms to append

reverse

Reverses a proper list.

Type Parameter Description
list lst the proper list to reverse

math.cspl

The math.cspl library contains Crisp functions that are designed for performing mathematical calculations and working with numbers.

Constants

This library defines the following constants:

Type Name Value Description
num pi 3.14159265358979323846 The ratio of the circumference of a circle to its diameter.

sum

Gives the sum total of all values in a given list.

Type Parameter Description
list x the list of numerical values

factorial

Calculates the factorial of a number.

Type Parameter Description
num x the number to calculate the factorial of

pow

Raises a number to a power.

Type Parameter Description
num x the base number
num e the exponent

square

Squares a number.

Type Parameter Description
num x the number to square

cube

Cubes a number.

Type Parameter Description
num x the number to cube

abs

Calculates the absolute value of a number.

Type Parameter Description
num x the number to calculate the absolute value of

nvc.cspl

The nvc.cspl library contains functions for working with name-value collections.

nvc-get

Gets the value from a name-value collection that corresponds to the specified name.

Type Parameter Description
list nvc the name-value collection
string name the name to retrieve the value for

querystring.cspl

The querystring.cspl library contains functions for working with query strings. Especially useful in web programming.

query-string-nvc

Gets the value from a name-value collection that corresponds to the specified name.

Type Parameter Description
list lst the list to convert

parse-query-string

Parses a query string.

Type Parameter Description
string str the string to parse

templating.cspl

The templating.cspl library contains functions for working with documenttemplates. Especially useful in web programming.

render-template

Substitutes a set of bracketed ({}) bindings in a document.

Type Parameter Description
string document the document to perform the replace operation on
list bindings the data bindings for the document

render-template-file

Substitutes a set of bracketed ({}) bindings in a document read from a file.

Type Parameter Description
string document the file path of the document to perform the replace operation on
list bindings the data bindings for the document
